Key Features

Model-Aware

Small models (Haiku/Sonnet) copy code from the knowledge base. Large models (Opus) enhance with animations, accessibility, dark mode, and SEO.

Knowledge Base

32 reference files with actual code for Next.js, Tailwind, shadcn/ui, Framer Motion, Prisma, Zod, Stripe, and more. Agents copy — not reinvent.

Token-Efficient

Project index system (.10x/) means agents read one JSON file instead of scanning 50+ source files. Saves tokens, saves money.

Global Components

39 component blueprints — navbar, sidebar, forms, cards, modals, tables, auth pages. Write once, reuse everywhere.

Feature Mapping

Every feature tracks which files, agents, and data flows are involved. Makes refactoring and debugging precise.

17 Quality Hooks

Lifecycle hooks enforce component reuse, block hardcoded secrets, auto-format code, and verify index updates.
